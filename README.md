# Dompetku Backend

Ini adalah backend untuk aplikasi Dompetku, dompet digital dengan fitur pulsa, kuota, token PLN, QRIS, dll.

## Cara Menjalankan
1. Install dependensi: `npm install`
2. Jalankan server: `npm start` atau `npm run dev`
